## О проекте

Текущий курс по Си имеет недостаточно практических заданий, к тому же почти одинаковых для студентов. В связи с этим есть необходимость доработки курса:

- На каждую тему необходимо сделать задачи трёх сложностей. Параметры задания (при наличии) должны генерироваться, исходя из ФИО студента;
- Студент должен в процессе выполнения заданий подготавливаться к выполнению курсовой работы;

Суть работы: курс строится в Moodle. Для тем по возможности используются старые записи лекций, а недостающую информацию нужно оформить в текстовом изложении. Задания с параметрами должны в идеале генерироваться вызовом общего метода, в параметрах которого указывается условное обозначение заданий (например, `theme_1_task_001`). Валидация ответа также должна проводиться через общий метод. Методы является частью интерфейса (=базового класса) генератора-валидатора. 

Содержимое курса в этом репозитории должно быть в виде файлов markdown.
