### Тема: Инициализация массива через указатель
**Сложность:** средняя

**Задание:**
Реализуйте функцию `{func_name}({type} *arr, int size, {type} k, int check_idx)`, которая последовательно заполняет переданный массив в цикле по формуле `*(arr + i) = i * k`. Обработка данных должна выполняться исключительно через указательную арифметику и явное разыменование, при этом использование синтаксиса прямой индексации `arr[i]` запрещено. По завершении заполнения необходимо вывести значение элемента с индексом `check_idx`, соблюдая формат `Value: {fmt}\n`.

**Формат вывода:** 
`Value: {fmt}\n`

**Уникальными значениями становятся:** `func_name`, `type`, `fmt`, `size`, `k`, `check_idx`
`seed % 2 == 0`: func_name = "fill_int_via_ptr", type = "int", fmt = "%d", size = 10, k = 7, check_idx = 4
`seed % 2 == 1`: func_name = "fill_float_via_ptr", type = "float", fmt = "%.1f", size = 12, k = 2.5, check_idx = 5

**Ввод:** Массив, размер, множитель и индекс передаются в функцию. Драйвер выделяет память, вызывает функцию и проверяет вывод.

**Пример решения:** (для seed=15)
```c
#include <stdio.h>

void fill_float_via_ptr(float *arr, int size, float k, int check_idx) {
    for (int i = 0; i < size; i++) {
        *(arr + i) = i * k;
    }
    printf("Value: %.1f\n", *(arr + check_idx));
}
```
