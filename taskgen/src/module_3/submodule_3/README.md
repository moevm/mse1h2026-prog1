# Подмодуль 3: Массивы

Подмодуль содержит 6 заданий по работе с массивами. 

## Темы заданий

 - объявление
 - инициализация
 - размер
 - массив и указатель – в чем разница
 - передача массива в функцию
 - многомерные массивы 


| № | Тема | Название |
|---|----------|------|
| 1 | Объявление | Объявление массива|
| 2 | Инициализация | Инициализация массива через указатель |
| 3 | Размер | Размер массива и указателя |
| 4 | Массив и указатель – в чем разница | Ошибка модификации имени массива |
| 5 | Передача массива в функцию | Работа с массивом через указатель в функции |
| 6 | Многомерные массивы | Преобразование дат через многомерный массив |
