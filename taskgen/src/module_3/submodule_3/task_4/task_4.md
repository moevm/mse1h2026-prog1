# Тема: "массив и указатель в чем разница?"
**Сложность:** средняя

## Задание:
В коде ниже есть ошибка. Найдите и исправьте её:

```c
int arr[5] = {10, 20, 30, 40, 50};
    for (i = 0; i < 5; i++) {
        printf("%d\n", *arr);
        arr++;
    }
```
Напишите исправленную версию кода, которая:
1. Объявляет указатель pa на начало массива
2. Обращается к элементам массива через указатель.
3. Выводит все элементы массива

## Формат вывода:
Element: 10
Element: 20
Element: 30
Element: 40
Element: 50


## Пример решения:
```c
#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *ptr = arr;
    int i;
    
    for (i = 0; i < 5; i++) {
        printf("Element: %d\n", *ptr);
        ptr++;
    }
    
    return 0;
}
```
