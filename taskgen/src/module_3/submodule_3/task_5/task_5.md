# Тема: "Передача массива в функцию"
**Сложность:** средняя
## Варианты задач (определяются по seed)

### seed % 2 == 0 : Задача 3.5.1
**Условие:**
Напишите функцию `int str_length(char *s)`, которая:
1. Принимает строку (массив char) через указатель
2. Использует арифметику указателей для подсчёта длины
3. Возвращает количество символов (без учёта `\0`)

**Формат вывода:**
`Length: X`

**Пример решения (str_length.c):**
```c
#include <stdio.h>

int str_length(char *s) {
    int n;
    for (n = 0; *s != '\0'; s++) {
        n++;
    }
    return n;
}
```

### seed % 2 == 1: Задача 3.5.2

**Условие:**
Напишите функцию `int array_sum(int *arr, int size)`, которая:
1. Принимает массив через указатель и его размер
2. Использует арифметику указателей для доступа к элементам массива
3. Возвращает сумму всех элементов

**Формат вывода:**
`Sum: X`

**Пример решения (array_sum.c):**
```c
#include <stdio.h>

int array_sum(int *arr, int size) {
    int sum = 0;
    int i;
    for (i = 0; i < size; i++) {
        sum += *(arr + i);
    }
    return sum;
}
