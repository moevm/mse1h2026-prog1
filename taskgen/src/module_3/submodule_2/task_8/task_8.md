# Тема: Указатель на указатель

### seed % 2 == 0 : 
**Условие:**
Напишите функцию `void redirect(int **ptr, int *new_target)`, которая:
1. Меняет адрес, на который указывает `*ptr`, через двойной указатель
2. После вызова `*ptr` должен указывать на `new_target`

**Пример решения:**
```c
#include <stdio.h>

void redirect(int **ptr, int *new_target) {
    *ptr = new_target;  
}
```

### seed % 2 == 1 :
**Условие:**
Напишите функцию `void swap_ptrs(int **a, int **b)`, которая:
1. Меняет местами два указателя (не значения, а именно адреса)
2. После вызова `*a` указывает на то, на что указывала `*b`, и наоборот

**Пример решения:**
```c
#include <stdio.h>

void swap_ptrs(int **a, int **b) {
    int *temp = *a;
    *a = *b;
    *b = temp;
}
```
