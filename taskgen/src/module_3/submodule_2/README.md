# Подмодуль 2: Указатели

Подмодуль содержит 8 заданий по работе с указателями. Задание 1, задание 2, задание 5 реализованы программно (на момент окончания итерации 2).

## Темы заданий

 - объявление
 - инициализация
 - null
 - разыменование
 - адрес переменной
 - арифметика указателей
 - сравнение указателей
 - указатель на указатель

| № | Тема | Название |
|---|----------|------|
| 1 | Объявление | Объявление указателя|
| 2 | Инициализация | Инициализация указателя адресом переменной |
| 3 | NULL | NULL: Поиск и исправление ошибки |
| 4 | Разыменовывание | Разыменовывание указателя |
| 5 | Адрес переменной | Работа с адресом через указатель |
| 6 | Арифметика указателей | Модификация элементов массива через арифметику указателей |
| 7 | Сравнение указателей | Сравнение указателей |
| 8 | Указатель на указатель | Изменение указателя через двойной указатель |
