#  Тема: Инициализация

**Сложность:** легкая

**Задание:**
Инициализируйте указатель ptr типа {type} переменной value.

**Уникальными значениями становятся: type:**
 seed % 3 == 0: type = float
 seed % 3 == 1: type = char
 seed % 3 == 2: type = int

**Ожидаемый вывод:** float *ptr = &value
