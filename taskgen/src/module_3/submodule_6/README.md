# Подмодуль 6: Указатель на функцию

Подмодуль содержит 3 задания по работе с указателями на функцию. 

## Темы заданий

 - синтаксис
 - передача функции как аргумент в функцию
 - таблицы функций



| № | Тема | Название |
|---|----------|------|
| 1 | синтаксис | Объявление указателя на функцию |
| 2 | передача функции как аргумент в функцию | Передача функции как аргумент |
| 3 | таблицы функций | Таблицы функций (массивы указателей) |
