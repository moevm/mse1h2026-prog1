# Подмодуль 5: Динамическая память

Подмодуль содержит 12 задания по работе с динамической памятью. 

## Темы заданий

 - malloc
 - calloc
 - realloc
 - free
 - проверка на null и как это важно
 - утечка памяти ( +++ напоминалка про санитайзер)
 - дабл фри
 - висячий указатель


| № | Тема | Название |
|---|----------|------|
| 1 | malloc, free | Выделение памяти через malloc |
| 2 | calloc, free | Выделение памяти через calloc |
| 3 | realoc, free | Изменение размера памяти через realloc |
| 4 - 7 | Библиотека stdlib.h | Функции динамической памяти |
| 8 | проверка на null и как это важно | 	
Классификация разыменования NULL по стандарту C |
| 9 | проверка на null и как это важно | Причины возврата NULL при выделении памяти |
| 10 | утечка памяти ( +++ напоминалка про санитайзер) | Утечка памяти и AddressSanitizer |
| 11 | дабл фри | Защита от Double Free |
| 12 | висячий указатель | Предотвращение висячего указателя |
