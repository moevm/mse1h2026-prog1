# Обработка ошибок: возврат кодов ошибок


### Задание №3
- **Сложность:** средне
- **Формулировка:**  
  Вам даны объявления двух функций и их коды возврата (соответствует вашему варианту).  
  Напишите функцию-обёртку `process`, которая:
  1. Вызывает первую функцию (`step1`). Если та вернула ненулевой код — немедленно возвращает `1`.
  2. Вызывает вторую функцию (`step2`) с результатом первой. Если та вернула ненулевой код — немедленно возвращает `2`.
  3. Если обе функции завершились успешно — записывает итоговый результат в `*out` и возвращает `0`.

  Писать `main()` не нужно.

- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Сигнатура `step1` | Сигнатура `step2` | Сигнатура `process` |
  |------------|-------------------|-------------------|----------------------|
  | 0 | `int parse_int(const char *s, int *out)` | `int double_val(int x, int *out)` | `int process(const char *s, int *out)` |
  | 1 | `int read_positive(int x, int *out)` | `int safe_sqrt(int x, int *out)` | `int process(int x, int *out)` |
  | 2 | `int clamp(int x, int limit, int *out)` | `int invert(int x, int *out)` | `int process(int x, int limit, int *out)` |
  | 3 | `int parse_double(const char *s, double *out)` | `int safe_log(double x, double *out)` | `int process(const char *s, double *out)` |

  Коды возврата каждой `step`-функции: `0` — успех, любое другое значение — ошибка.  
  Коды возврата `process`: `0` — успех, `1` — ошибка на шаге 1, `2` — ошибка на шаге 2.
