# Подмодуль 6: Обработка ошибок в C


Подмодуль содержит 3 задания по работе с механизмами обработки ошибок в C.


## Темы заданий


| № | Название | Тема |
|---|----------|------|
| 1 | Обработка ошибок: errno | Определение числового значения `errno` после неудачного системного вызова |
| 2 | Обработка ошибок: perror | Сопоставление фрагментов кода с выводом `perror` в `stderr` |
| 3 | Обработка ошибок: возврат кодов ошибок | Написание функции-обёртки с цепочкой вызовов и ранним возвратом кода ошибки |
