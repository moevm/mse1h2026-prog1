# Обработка ошибок: errno


### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Вам дан фрагмент кода на C и описание условий его выполнения (соответствует вашему варианту).  
  Определите, какое числовое значение будет у переменной `saved_err` после выполнения этого кода на платформе Linux (x86-64).  
  Введите одно целое число.


- **Параметры задания:**  
  Вариант определяется значением `seed % 2`:

  | `seed % 2` | Условие выполнения | Правильный ответ |
  |------------|--------------------|------------------|
  | 0 | Файл `data.txt` не существует | `2` (`ENOENT`) |
  | 1 | Файл `/root/secret.txt` существует, но у процесса нет прав на чтение | `13` (`EACCES`) |

  Фрагмент кода (одинаковый для всех вариантов, меняется только имя/путь файла):

  | `seed % 2` | Фрагмент кода |
  |------------|---------------|
  | 0 | `filename = "data.txt"` |
  | 1 | `filename = "/root/secret.txt"` |

  ```c
  #include <stdio.h>
  #include <errno.h>

  int main(void) {
      errno = 0;
      FILE *f = fopen("<filename>", "r");
      int saved_err = errno;
      if (f) fclose(f);
      return 0;
  }
  ```
