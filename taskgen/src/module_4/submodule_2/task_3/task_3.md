# Union: отличие от struct

### Задание №3
- **Сложность:** легко  
- **Формулировка:**  
  Объявите структуру `S` и объединение `U` с одинаковым набором полей (набор указан ниже).  
  Напишите программу, которая выводит их размеры в байтах в формате:
  ```
  struct: N
  union: M
  ```
  Используйте оператор `sizeof`.

- **Параметры задания:**  
Набор полей определяется значением `seed % 5`:

| `seed % 5` | Поля | `sizeof(struct S)` | `sizeof(union U)` |
|------------|------|--------------------|-------------------|
| 0 | `char a; int b; double c;` | 16 | 8 |
| 1 | `int a; float b; char c[4];` | 12 | 4 |
| 2 | `char a; char b; short c;` | 4 | 2 |
| 3 | `short a; long b; float c;` | 24 | 8 |
| 4 | `int a; int b; char c[8];` | 16 | 8 |
