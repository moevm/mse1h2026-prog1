# Union: отличие от struct

### Задание №3
- **Сложность:** легко  
- **Формулировка:**  
  Объявите структуру `S` и объединение `U` с одинаковым набором полей (набор соответствует вашему варианту).  
  Напишите функцию `get_sizes`, которая возвращает через выходные параметры размеры `struct S` и `union U`, вычисленные оператором `sizeof`.

  Подключать заголовочные файлы не требуется.  
  Писать `main` не нужно — только объявления `struct S`, `union U` и тело функции.  
  Сигнатура функции: `void get_sizes(size_t *s_size, size_t *u_size)`.  
  Тестирующая система вызовет функцию и проверит возвращённые значения.

- **Параметры задания:**  
  Набор полей определяется значением `seed % 5`:

  | `seed % 5` | Поля `struct S` и `union U` |
  |------------|-----------------------------|
  | 0 | `char a; int b; double c;` |
  | 1 | `int a; float b; char c[4];` |
  | 2 | `char a; char b; short c;` |
  | 3 | `short a; long b; float c;` |
  | 4 | `int a; int b; char c[8];` |
