# Подмодуль 2: Объединения (union) в C

Подмодуль содержит 3 задания по работе с объединениями. 

## Темы заданий

| № | Название | Тема |
|---|----------|------|
| 1 | Union: использование | Чтение тега и значения, запись в соответствующее поле union, вывод |
| 2 | Union: перекрытия памяти | Интерпретация 32-битного целого как массива байт (порядок байт little-endian) |
| 3 | Union: отличие от struct | Сравнение размеров структуры и объединения с одинаковыми полями |
