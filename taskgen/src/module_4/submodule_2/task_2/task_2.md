# Union: перекрытия памяти

### Задание №2
- **Сложность:** средне  
- **Формулировка:**  
  Дано объявление:
  ```c
  #include <stdint.h>

  union ByteView {
      uint32_t full;
      unsigned char bytes;
  };
  ```

  Напишите функцию `print_bytes`, которая:
  1. Принимает значение типа `uint32_t`.
  2. Записывает его в поле `full` объекта `union ByteView`.
  3. Выводит все 4 байта представления числа в памяти через пробел в шестнадцатеричном формате (`%02x`), начиная с `bytes[0]`.

  Подключать заголовочные файлы не требуется.  
  Писать `main` не нужно — только тело функции.  
  Сигнатура функции: `void print_bytes(uint32_t value)`.  
  Тестирующая система вызовет функцию с различными значениями и проверит вывод.

- **Пример:**

  Функция вызвана как `print_bytes(0x12345678)`.

  **Ожидаемый вывод:**
  ```
  78 56 34 12
  ```
