# Union: перекрытия памяти

### Задание №2
- **Сложность:** средне  
- **Формулировка:**  
  Дано объявление:
  ```c
  #include <stdint.h>

  union ByteView {
      uint32_t full;
      unsigned char bytes[4];
  };
  ```

  Напишите программу, которая:
  1. Присваивает полю `full` значение `<hex_value>`.
  2. Выводит все 4 байта представления числа в памяти через пробел в шестнадцатеричном формате (`%02x`), начиная с `bytes[0]`.

- **Параметры задания:**  
  Значение `full` определяется по `seed % 5`:

  | `seed % 5` | Значение `full` (<hex_value>) | Ожидаемый вывод |
  |------------|------------------------|--------------------------------|
  | 0          | `0x12345678`           | `78 56 34 12`                  |
  | 1          | `0xDEADBEEF`           | `ef be ad de`                  |
  | 2          | `0xCAFEBABE`           | `be ba fe ca`                  |
  | 3          | `0x00FF00FF`           | `ff 00 ff 00`                  |
  | 4          | `0xABCD1234`           | `34 12 cd ab`                  |

- **Пример (для `seed = 5`, `seed % 5 = 0` → `0x12345678`):**  

  **Входные данные:** отсутствуют  

  **Ожидаемый вывод:**

  ```
  78 56 34 12
  ```
