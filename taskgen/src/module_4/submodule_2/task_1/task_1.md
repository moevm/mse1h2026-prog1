# Union: использование

### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Дано объявление union (соответствует вашему варианту).
   ```c
  union <Имя> {
      int <Целое поле>;
      <тип> <Вещественное поле>;
  };
  ```
  Напишите программу, которая:
  1. Считывает из стандартного ввода строку вида:  
     `<тег> <значение>` (тег — символ, значение — целое или вещественное число).
  2. Записывает значение в соответствующее поле union в зависимости от тега.
  3. Выводит значение в формате, соответствующем типу поля.

  Формат вывода:  
  - если тег соответствует целому полю — `%d`;  
  - если тег соответствует вещественному полю — `%.2f`.

- **Параметры задания:**  
  Вариант задания определяется значением `seed % 4`:

  | `seed % 4` | <Имя> union | <Целое поле> (тип) | <Вещественное поле> (<тип>) | Тег int | Тег float |
  |------------|-----------|------------------|-------------------------|---------|-----------|
  | 0 | `Value` | `count` (`int`) | `measurement` (`double`) | `'i'` | `'d'` |
  | 1 | `Payload` | `code` (`int`) | `ratio` (`float`) | `'i'` | `'f'` |
  | 2 | `Reading` | `raw` (`int`) | `voltage` (`float`) | `'n'` | `'v'` |
  | 3 | `Data` | `flags` (`int`) | `rate` (`double`) | `'i'` | `'d'` |

  Числовые значения для тестов генерируются на основе `seed`:  
  - целое: `int_val = 10 + (seed % 90)`  
  - вещественное: `flt_val = (seed % 100) / 10.0 + 1.0` (выводится с двумя знаками после запятой)

- **Пример (для `seed = 4`, `seed % 4 = 0` → `Value`, `int_val = 14`, `flt_val = 1.40`):**
- 
  **Тест 1**
  
  **Входные данные:**
  ```
  i 14
  ```
  
  **Выходные данные:**
  ```
  14
  ```

  **Тест 2**
  
  **Входные данные:**
  ```
  d 1.40
  ```
  
  **Выходные данные:**
  ```
  1.40
  ```
