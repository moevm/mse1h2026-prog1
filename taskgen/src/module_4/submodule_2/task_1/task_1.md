# Union: использование

### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Дано объявление `union` (соответствует вашему варианту):
  ```c
  union <Имя> {
      int <Целое поле>;
      <тип> <Вещественное поле>;
  };
  ```
  Напишите функцию `fill_<UnionName>`, которая:
  1. Принимает тег (`char tag`) и числовое значение (`double value`).
  2. В зависимости от тега записывает значение в **соответствующее поле** union:
     - если тег соответствует целому полю — преобразует `value` к `int` и записывает в целое поле;
     - если тег соответствует вещественному полю — преобразует `value` к типу вещественного поля и записывает в него.
  3. Возвращает заполненный `union <Имя>`.

  Подключать заголовочные файлы не требуется.  
  Писать `main`, считывание данных и вывод результата не нужно — только тело функции.  
  **Сигнатура функции задана и соответствует вашему варианту** (см. таблицу).  
  Тестирующая система вызовет функцию и проверит поля возвращённого union напрямую.

- **Параметры задания:**  
  Вариант задания определяется значением `seed % 4`:

  | `seed % 4` | Имя union | Целое поле (тип) | Вещественное поле (тип) | Тег int | Тег float | Сигнатура функции |
  |------------|-----------|-----------------|------------------------|---------|-----------|-------------------|
  | 0 | `Value` | `count` (`int`) | `measurement` (`double`) | `'i'` | `'d'` | `union Value fill_Value(char tag, double value)` |
  | 1 | `Payload` | `code` (`int`) | `ratio` (`float`) | `'i'` | `'f'` | `union Payload fill_Payload(char tag, double value)` |
  | 2 | `Reading` | `raw` (`int`) | `voltage` (`float`) | `'n'` | `'v'` | `union Reading fill_Reading(char tag, double value)` |
  | 3 | `Data` | `flags` (`int`) | `rate` (`double`) | `'i'` | `'d'` | `union Data fill_Data(char tag, double value)` |

  Числовые значения, с которыми будет вызвана функция, генерируются на основе `seed`:
  - целое: `int_val = 10 + (seed % 90)`
  - вещественное: `flt_val = (seed % 100) / 10.0 + 1.0`

- **Пример для `seed = 4` (`seed % 4 = 0` → `Value`, `int_val = 14`, `flt_val = 1.4`):**

  **Тест 1** — функция вызвана как `fill_Value('i', 14.0)`.  
  Ожидается: возвращённый union имеет `count == 14`.

  **Тест 2** — функция вызвана как `fill_Value('d', 1.4)`.  
  Ожидается: возвращённый union имеет `measurement ≈ 1.40`.
