# Struct: выравнивание (padding)

### Задание №5
- **Сложность:** средне  
- **Формулировка:**  
  Дана структура (платформа x86-64, компилятор gcc):  
  ```c
  struct S {
      поля
  };
  ```
  Сколько байт padding (выравнивающих пустот) добавляет компилятор в эту структуру?
  Введите целое число.

- **Параметры задания:**  
  Набор полей определяется значением `seed % 5`:

  | `seed % 5` | Поля | Верный ответ |
  |------------|------|---------|
  | 0 | `char a; int b;` | 3 |
  | 1 | `char a; short b; double c;` | 5 |
  | 2 | `int a; char b; int c;` | 3 |
  | 3 | `char a; char b; int c; double d;` | 2 |
  | 4 | `short a; char b; double c; int d;` | 9 |
