# Stcruct: объявление структуры

### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Объявите структуру, описывающую сущность согласно вашему варианту.  
  Структура должна содержать поля, перечисленные в таблице ниже.  
  Напишите только объявление типа структуры (с `typedef` или без).  
  Подключать заголовочные файлы и писать функцию `main()` не нужно.  
- **Параметры задания:**  
  В зависимости от значения `seed` выбирается сущность и набор полей:

  | `seed % 5` | Имя структуры | Сущность       | Поля                                                         |
  |------------|---------------|----------------|--------------------------------------------------------------|
  | 0          | `Student`     | студент университета | `char name[32]`, `int age`, `float gpa`                      |
  | 1          | `Book`        | книга в библиотеке   | `char title[64]`, `int pages`, `double price`                |
  | 2          | `Point2D`     | точка на плоскости   | `double x`, `double y`, `int label`                          |
  | 3          | `Rectangle`   | прямоугольник         | `double width`, `double height`, `char color[16]`            |
  | 4          | `Employee`    | сотрудник компании    | `char name[32]`, `int id`, `double salary`                   |
