# Struct: доступ к полям

### Задание №3
- **Сложность:** средне  
- **Формулировка:**  
  Вам дано объявление структуры (соответствует вашему варианту).  
  Напишите функцию `compute_<TypeName>`, которая:
  1. Принимает массив структур указанного типа и его длину `n`.
  2. Вычисляет и возвращает результат операции `op` над целевым полем всех элементов массива.

  Подключать заголовочные файлы не требуется.  
  Писать `main`, считывание данных и вывод результата не нужно — только тело функции.

  **Сигнатура функции задана и соответствует вашему варианту** (см. таблицу).  
  Функция будет вызвана автоматически с заранее заполненным массивом; результат будет выведен тестирующей системой.

- **Параметры задания:**  
  Вариант задания определяется значением `seed`.

  - **Операция** `op = seed % 3`:
    - `0` — найти **максимум** значений целевого поля.
    - `1` — найти **сумму** значений целевого поля.
    - `2` — найти **минимум** значений целевого поля.

  | `seed % 5` | Тип структуры | Объявление структуры (дано в условии) | Целевое поле (тип) | Сигнатура функции |
  |------------|---------------|---------------------------------------|--------------------|-------------------|
  | 0 | `Student` | `typedef struct { char name[32]; int age; float gpa; } Student;` | `age` (`int`) | `int compute_Student(Student arr[], int n)` |
  | 1 | `Book` | `typedef struct { char title[64]; int pages; double price; } Book;` | `pages` (`int`) | `int compute_Book(Book arr[], int n)` |
  | 2 | `Point2D` | `typedef struct { double x; double y; int label; } Point2D;` | `label` (`int`) | `int compute_Point2D(Point2D arr[], int n)` |
  | 3 | `Rectangle` | `typedef struct { double width; double height; char color[16]; } Rectangle;` | `width` (`double`) | `double compute_Rectangle(Rectangle arr[], int n)` |
  | 4 | `Employee` | `typedef struct { char name[32]; int id; double salary; } Employee;` | `salary` (`double`) | `double compute_Employee(Employee arr[], int n)` |

  Длина массива фиксирована: `n = 3`.

- **Пример для `seed = 7` (`seed % 5 = 2` → `Point2D`, `op = 1` → сумма `label`):**

  Функция будет вызвана как `compute_Point2D(arr, 3)`, где массив содержит объекты:
  ```
  { .x=1.0, .y=2.0, .label=10 }
  { .x=3.5, .y=4.5, .label=20 }
  { .x=0.0, .y=1.0, .label=15 }
  ```

  Ожидаемый результат: `45`.
