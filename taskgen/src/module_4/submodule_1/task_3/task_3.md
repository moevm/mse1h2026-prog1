# Struct: доступ к полям

### Задание №3
- **Сложность:** средне  
- **Формулировка:**  
  Вам дано объявление структуры (соответствует вашему варианту).  
  Напишите программу, которая:
  1. Считывает из стандартного ввода данные о 3 объектах указанного типа (каждая строка содержит значения полей в заданном порядке через пробел).
  2. Сохраняет объекты в массив структур.
  3. Вычисляет и выводит результат операции `op` над одним из полей.

  Формат вывода:`format`.

- **Параметры задания:**  
  Вариант задания определяется значением `seed`.  
  Параметризуются:
  - **Тип структуры** (`seed % 5`), порядок полей при вводе и целевое поле для операции.
  - **Операция** `op = seed % 3`:
    - `0` – найти **максимум** значений поля.
    - `1` – найти **сумму** значений поля.
    - `2` – найти **минимум** значений поля.

  | `seed % 5` | Имя структуры | Объявление структуры (дано в условии) | Порядок полей при вводе | Целевое поле (тип) | Формат вывода (`format`)  |
  |------------|---------------|---------------------------------------|-------------------------|-------------------|-------------------|
  | 0 | `Student` | `typedef struct { char name[32]; int age; float gpa; } Student;` | `name`, `age`, `gpa` | `age` (`int`) | `%d`
  | 1 | `Book` | `typedef struct { char title[64]; int pages; double price; } Book;` | `title`, `pages`, `price` | `pages` (`int`) | `%d`
  | 2 | `Point2D` | `typedef struct { double x; double y; int label; } Point2D;` | `x`, `y`, `label` | `label` (`int`) | `%d`
  | 3 | `Rectangle` | `typedef struct { double width; double height; char color[16]; } Rectangle;` | `width`, `height`, `color` | `width` (`double`) | `%.1f`
  | 4 | `Employee` | `typedef struct { char name[32]; int id; double salary; } Employee;` | `name`, `id`, `salary` | `salary` (`double`) | `%.1f`


- **Пример (для `seed = 7`, `seed % 5 = 2` → `Point2D`, `n = 3`, `op = 1` → сумма `label\):**

**Входные данные:**
```
1.0 2.0 10
3.5 4.5 20
0.0 1.0 15
```

**Выходные данные:**
```
45
```
