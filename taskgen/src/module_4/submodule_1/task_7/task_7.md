# Struct: sizeof(struct)

### Задание №7
- **Сложность:** средне  
- **Формулировка:**  
  Дана структура (платформа x86-64, компилятор gcc):  
  ```c
  struct S {
      поля
  };
  ```
  Чему равен `sizeof(struct S)`?   
  Введите целое число.

- **Параметры задания:**  
  Набор полей определяется значением `seed % 5`:

  | `seed % 5` | Поля | Верный ответ (`sizeof`) |
  |------------|------|-------------------------|
  | 0 | `char a; int b;` | 8 |
  | 1 | `char a; short b; double c;` | 16 |
  | 2 | `int a; char b; int c;` | 12 |
  | 3 | `char a; char b; int c; double d;` | 16 |
  | 4 | `short a; char b; double c; int d;` | 24 |
