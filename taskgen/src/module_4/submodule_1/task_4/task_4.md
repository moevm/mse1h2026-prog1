# Struct: выравнивание (alignment)

### Задание №4
- **Сложность:** легко  
- **Формулировка:**  
  Дана структура (платформа x86-64, компилятор gcc):  
  ```c
  struct S {
      char a;
      TYPE b;
      char c;
  };
  ```
  По какому смещению (в байтах от начала структуры) расположено поле `b`?  
  Введите целое число (смещение).  

- **Параметры задания:**  
  Тип поля `b` определяется значением `seed % 4`:

  | `seed % 4` | `TYPE` | Верный ответ |
  |------------|--------|--------------|
  | 0          | `short` | 2            |
  | 1          | `int`   | 4            |
  | 2          | `float` | 4            |
  | 3          | `double`| 8            |
