# stdlib: FILE


### Задание №1
- **Сложность:** легко
- **Формулировка:**  
  Напишите **одну строку** — объявление переменной типа `FILE *` с именем `fp`
  и её инициализацией значением, соответствующим вашему варианту.

  Подключать заголовочные файлы и писать `main()` не нужно.


- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Инициализатор | Пояснение |
  |------------|---------------|-----------|
  | 0 | `NULL` | нулевой (не открытый) файловый указатель |
  | 1 | `stdin` | стандартный ввод |
  | 2 | `stdout` | стандартный вывод |
  | 3 | `stderr` | стандартный вывод ошибок |
