# Подмодуль 5: Стандартная библиотека (stdlib) в C


Подмодуль содержит 12 заданий по работе с функциями стандартной библиотеки `<stdlib.h>` и `<stdio.h>`.


## Темы заданий


| № | Название | Тема |
|---|----------|------|
| 1 | stdlib: FILE | Объявление и инициализация файлового указателя `FILE *` |
| 2 | stdlib: fopen | Открытие файла с помощью `fopen` в заданном режиме |
| 3 | stdlib: fclose | Закрытие файла с помощью `fclose` |
| 4 | stdlib: fread | Чтение бинарных данных из файла в массив через `fread` |
| 5 | stdlib: fwrite | Запись бинарных данных из массива в файл через `fwrite` |
| 6.1 | stdlib: текстовый режим | Запись и чтение одного значения через `fprintf`/`fscanf` |
| 6.2 | stdlib: бинарный режим | Запись и чтение одного значения через `fwrite`/`fread` |
| 7 | stdlib: qsort | Сортировка массива структур по заданному полю с использованием `qsort` |
| 8 | stdlib: bsearch | Бинарный поиск в отсортированном массиве структур с использованием `bsearch` |
| 9 | stdlib: exit — успешное завершение | Завершение программы с кодом `EXIT_SUCCESS` через `exit` |
| 9.2 | stdlib: exit — завершение с ошибкой | Завершение программы с кодом `EXIT_FAILURE` через `exit` |
| 10 | stdlib: atexit | Порядок вызова функций, зарегистрированных через `atexit` |
| 11 | stdlib: getenv | Чтение переменной окружения через `getenv` с преобразованием типа |
| 12 | stdlib: system | Передача команды командному процессору через `system` |
