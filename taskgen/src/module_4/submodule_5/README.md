# Подмодуль 5: Стандартная библиотека (qsort, bsearch) в C

Подмодуль содержит 2 задания по использованию функций `qsort` и `bsearch` для сортировки и бинарного поиска в массивах структур. Все задания на момент ококнчания второй итерации написаны только текстово.

## Темы заданий

| № | Название | Тема |
|---|----------|------|
| 7 | stdlib: qsort | Сортировка массива структур по заданному полю с использованием `qsort` |
| 8 | stdlib: bsearch | Бинарный поиск в отсортированном массиве структур с использованием `bsearch` |
