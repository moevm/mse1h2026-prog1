# stdlib: system


### Задание №12
- **Сложность:** легко
- **Формулировка:**  
  Напишите **одну строку** на C — вызов стандартной библиотечной функции,
  которая передаёт команду командному процессору операционной системы
  и возвращает код её завершения.  
  Результат сохраните в переменную `ret` (уже объявлена).


- **Параметры задания:**  
  Вариант определяется значением `seed % 4`.  

  | `seed % 4` | Назначение команды | Итоговая строка |
  |------------|--------------------|-----------------|
  | 0 | Вывести текущую дату и время | `ret = system("date");` |
  | 1 | Вывести список файлов текущей директории | `ret = system("ls");` |
  | 2 | Вывести имя текущего пользователя | `ret = system("whoami");` |
  | 3 | Вывести путь к текущей рабочей директории | `ret = system("pwd");` |
