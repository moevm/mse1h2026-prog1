# stdlib: bsearch

### Задание №8
- **Сложность:** средне  
- **Формулировка:**  
  Вам дано объявление структуры (соответствует вашему варианту).  
  Массив из `N` элементов уже отсортирован по полю **key_field** (в порядке возрастания).  
  Напишите программу, которая:
  1. Считывает из стандартного ввода значение ключа поиска.
  2. С помощью функции `bsearch` находит элемент с таким значением поля **key_field**.
  3. Выводит все поля найденного элемента в том же порядке, в котором они перечислены в структуре (через пробел).  
     Если элемент не найден — выводит `"Not found"`.

  Используйте функцию сравнения, совместимую с `bsearch`.  

- **Параметры задания:**  
  Вариант задания определяется значением `seed`.  
  Параметризуются:
  - **Тип структуры** (`seed % 5`), порядок полей и ключевое поле для поиска.

  | `seed % 5` | Имя структуры | Поля (порядок вывода) | Ключевое поле (тип) | N | Массив (значения ключа) | Формат вывода |
  |------------|---------------|-----------------------|---------------------|---|-------------------------|---------------|
  | 0 | `Student` | `char name[32]`, `int age`, `float gpa` | `age` (`int`) | 5 | `{18, 19, 20, 21, 22}` | `%s %d %.1f` |
  | 1 | `Book` | `char title[64]`, `int pages`, `double price` | `pages` (`int`) | 5 | `{150, 200, 250, 300, 350}` | `%s %d %.2f` |
  | 2 | `Point2D` | `double x`, `double y`, `int label` | `label` (`int`) | 4 | `{10, 20, 30, 40}` | `%.1f %.1f %d` |
  | 3 | `Rectangle` | `double width`, `double height`, `char color[16]` | `width` (`double`) | 4 | `{1.5, 2.0, 2.5, 3.0}` | `%.1f %.1f %s` |
  | 4 | `Employee` | `char name[32]`, `int id`, `double salary` | `salary` (`double`) | 5 | `{35000, 42000, 50000, 58000, 65000}` | `%s %d %.2f` |

- **Пример (для `seed = 2`, `seed % 5 = 2` → `Point2D`):**  

  **Тест 1:**  
  Ввод:
  ```
  20
  ```
  Вывод:
  ```
  2.0 3.0 20
  ```
  **Тест 2:**  
  Ввод:
  ```
  99
  ```
  Вывод:
  ```
  Not found
  ```
