# stdlib: fread


### Задание №4
- **Сложность:** средне
- **Формулировка:**  
  Вам дана сигнатура функции (соответствует вашему варианту).  
  Напишите функцию `read_items`, которая:
  1. Открывает файл с именем `filename` в бинарном режиме для чтения.
  2. Считывает не более `count` элементов в массив `buf`.
  3. Закрывает файл.
  4. Возвращает фактическое количество прочитанных элементов.  
  Если файл не удалось открыть — возвращает `0`.

  Писать `main()` не нужно.


- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Сигнатура функции | `count` |
  |------------|-------------------|-----------------|
  | 0 | `int read_items(const char *filename, int *buf, int count)` | 5 |
  | 1 | `int read_items(const char *filename, double *buf, int count)` | 3 |
  | 2 | `int read_items(const char *filename, float *buf, int count)` | 8 |
  | 3 | `int read_items(const char *filename, short *buf, int count)` | 6 |
