# stdlib: fopen


### Задание №2
- **Сложность:** легко
- **Формулировка:**  
  Напишите **одну строку** — вызов `fopen` с присваиванием результата
  переменной `FILE *fp`, соответствующий вашему варианту.

  Имя файла и режим открытия определяются вашим вариантом (см. таблицу).  
  Подключать заголовочные файлы и писать `main()` не нужно. 


- **Параметры задания:**  
  Вариант определяется значением `seed % 6`:

  | `seed % 6` | Имя файла | Режим | Пояснение |
  |------------|-----------|-------|-----------|
  | 0 | `"data.txt"` | `"r"` | открыть текстовый файл для чтения |
  | 1 | `"out.txt"` | `"w"` | открыть текстовый файл для записи |
  | 2 | `"log.txt"` | `"a"` | открыть текстовый файл для дозаписи в конец |
  | 3 | `"data.bin"` | `"rb"` | открыть бинарный файл для чтения |
  | 4 | `"out.bin"` | `"wb"` | открыть бинарный файл для записи |
  | 5 | `"data.txt"` | `"r+"` | открыть текстовый файл для чтения и записи |
