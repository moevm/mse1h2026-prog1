# stdlib: fclose


### Задание №3
- **Сложность:** легко
- **Формулировка:**  
  Напишите **одну строку** — вызов `fclose` для переменной,
  соответствующей вашему варианту.


- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Имя переменной | Чем открыта |
  |------------|----------------|-------------|
  | 0 | `fp` | `fopen("data.txt", "r")` |
  | 1 | `fin` | `fopen("input.bin", "rb")` |
  | 2 | `fout` | `fopen("output.txt", "w")` |
  | 3 | `flog` | `fopen("log.txt", "a")` |
