# stdlib: getenv


### Задание №11
- **Сложность:** средне
- **Формулировка:**  
  Напишите функцию `get_env_value`, которая считывает переменную окружения
  с именем `name`, преобразует её значение к нужному типу и возвращает результат.  
  Если переменная не задана — возвращает значение по умолчанию `default_val`.

  Сигнатура и логика преобразования определяются вашим вариантом.  
  Писать `main()` не нужно.


- **Параметры задания:**  
  Вариант определяется значением `seed % 3`:

  | `seed % 3` | Сигнатура | Переменная окружения | Преобразование | Значение по умолчанию |
  |------------|-----------|----------------------|----------------|-----------------------|
  | 0 | `int get_env_value(const char *name, int default_val)` | `"TIMEOUT"` | `atoi` | `30` |
  | 1 | `double get_env_value(const char *name, double default_val)` | `"THRESHOLD"` | `atof` | `0.5` |
  | 2 | `long get_env_value(const char *name, long default_val)` | `"MAX_SIZE"` | `atol` | `1024` |
