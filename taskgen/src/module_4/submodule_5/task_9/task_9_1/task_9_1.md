# stdlib: exit — успешное завершение


### Задание №9
- **Сложность:** легко
- **Формулировка:**  
  Напишите **одну строку** — вызов `exit` для штатного завершения программы.

  Используйте именованный макрос из `<stdlib.h>`, а не числовую константу.  
  Подключать заголовочные файлы и писать `main()` не нужно.

- **Ожидаемый ответ:**
  ```c
  exit(EXIT_SUCCESS);
  ```
