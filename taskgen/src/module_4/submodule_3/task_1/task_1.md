# Enum: значения по умолчанию

### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Дано объявление перечисления (соответствует вашему варианту).  
  Напишите функцию `next_<EnumName>`, которая:
  1. Принимает значение типа соответствующего `enum`.
  2. Возвращает **следующий элемент** перечисления по кругу (после последнего идёт первый).

  Подключать заголовочные файлы не требуется.  
  Писать `main` не нужно — только объявление `enum` и тело функции.  
  Использовать `if/else` или `switch` **не допускается** — решение должно использовать арифметику над целыми значениями enum.  
  **Сигнатура функции задана и соответствует вашему варианту** (см. таблицу). 

- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Имя enum | Члены (значения по умолчанию: 0, 1, 2, 3) | Сигнатура функции |
  |------------|----------|--------------------------------------------|-------------------|
  | 0 | `Direction` | `NORTH`, `EAST`, `SOUTH`, `WEST` | `enum Direction next_Direction(enum Direction d)` |
  | 1 | `Season` | `SPRING`, `SUMMER`, `AUTUMN`, `WINTER` | `enum Season next_Season(enum Season d)` |
  | 2 | `Suit` | `CLUBS`, `DIAMONDS`, `HEARTS`, `SPADES` | `enum Suit next_Suit(enum Suit d)` |
  | 3 | `Channel` | `RED`, `GREEN`, `BLUE`, `ALPHA` | `enum Channel next_Channel(enum Channel d)` |

  Количество элементов во всех вариантах: 4.

- **Пример для `seed = 8` (`seed % 4 = 0` → `Direction`):**

  - `next_Direction(NORTH)` → `EAST`
  - `next_Direction(WEST)` → `NORTH`
  - `next_Direction(SOUTH)` → `WEST`
