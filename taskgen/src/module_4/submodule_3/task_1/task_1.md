# Enum: значения по умолчанию

### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Дано объявление перечисления (enum) (соответствует вашему варианту).  
  Напишите программу, которая считывает из стандартного ввода целое число `n` и выводит имя соответствующего элемента перечисления.  
  Если `n` не соответствует ни одному элементу — выводится `unknown`.  

- **Параметры задания:**  
  Тема перечисления определяется значением `seed % 4`:

  | `seed % 4` | Имя enum | Члены (порядок = значение) |
  |------------|----------|----------------------------|
  | 0 | `Direction` | `NORTH`, `EAST`, `SOUTH`, `WEST` |
  | 1 | `Season` | `SPRING`, `SUMMER`, `AUTUMN`, `WINTER` |
  | 2 | `Suit` | `CLUBS`, `DIAMONDS`, `HEARTS`, `SPADES` |
  | 3 | `Channel` | `RED`, `GREEN`, `BLUE`, `ALPHA` |


- **Пример (для `seed = 8`, `seed % 4 = 0` → `Direction`):**  

  **Входные данные (тест 1):**
  ```
  0
  ```
  **Вывод:**
  ```
  NORTH
  ```
    **Входные данные (тест 2):**
  ```
  13
  ```
  **Вывод:**
  ```
  unknown
  ```
