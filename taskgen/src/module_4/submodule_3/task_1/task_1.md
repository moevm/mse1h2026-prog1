# Enum: значения по умолчанию

### Задание №1
- **Сложность:** легко
- **Формулировка:**  
  Напишите функцию `next_<EnumName>`, которая:
  1. Принимает значение типа `enum <EnumName>`.
  2. Возвращает следующий элемент перечисления по кругу (после последнего идёт первый).

  Объявление `enum` и `main` писать не нужно — они предоставляются тестирующей системой.  
  Подключать заголовочные файлы не требуется.  
  **Сигнатура функции задана и соответствует вашему варианту** (см. таблицу).  
  Количество элементов во всех вариантах: **4**.  
  Конкретные члены перечисления не известны — решение должно опираться только на их числовые значения.

- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Имя enum | Сигнатура функции | Члены enum [скрыто от студента] |
  |------------|----------|-------------------|---------------------------------|
  | 0 | `Direction` | `enum Direction next_Direction(enum Direction d)` | `NORTH=0, EAST=1, SOUTH=2, WEST=3` |
  | 1 | `Season` | `enum Season next_Season(enum Season d)` | `SPRING=0, SUMMER=1, AUTUMN=2, WINTER=3` |
  | 2 | `Suit` | `enum Suit next_Suit(enum Suit d)` | `CLUBS=0, DIAMONDS=1, HEARTS=2, SPADES=3` |
  | 3 | `Channel` | `enum Channel next_Channel(enum Channel d)` | `RED=0, GREEN=1, BLUE=2, ALPHA=3` |
