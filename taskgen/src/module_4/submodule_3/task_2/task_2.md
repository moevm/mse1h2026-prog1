# Enum: явное задание значений

### Задание №2
- **Сложность:** средне  
- **Формулировка:**  
  Объявите перечисление `{{ enum_name }}` со следующими членами и явными значениями:
  ```
  {{ members_list }}
  ```
  Напишите программу, которая считывает из стандартного ввода целое число и выводит имя соответствующего члена перечисления или `unknown`, если значение не совпадает ни с одним.
  
  - **Параметры задания:**  
  Тема перечисления определяется значением `seed % 4`:
  
  | `seed % 4` | Имя enum | Члены и явные значения |
  |------------|----------|------------------------|
  | 0 | `HttpStatus` | `OK=200, FORBIDDEN=403, NOT_FOUND=404, SERVER_ERROR=500` |
  | 1 | `Priority` | `LOW=1, MEDIUM=5, HIGH=10, CRITICAL=100` |
  | 2 | `Permission` | `NONE=0, EXEC=1, WRITE=2, READ=4` |
  | 3 | `LogLevel` | `DEBUG=10, INFO=20, WARN=30, ERROR=40` |
  
  - **Пример (для `seed = 3`, `seed % 4 = 3` → `LogLevel`):**  

  **Тест 1:**  
  Ввод:
  ```
  30
  ```
  Вывод:
  ```
  WARN
  ```
  **Тест 2:**  
  Ввод:
  ```
  25
  ```
  Вывод:
  ```
  unknown
  ```
