# Подмодуль 3: Перечисления (enum) в C

Подмодуль содержит 2 задания по работе с перечислениями. 

## Темы заданий

| № | Название | Тема |
|---|----------|------|
| 1 | Enum: значения по умолчанию | Сопоставление целого числа с элементом перечисления (порядковые значения от 0) |
| 2 | Enum: явное задание значений | Сопоставление целого числа с элементом перечисления с явно заданными значениями |
