# Подмодуль 7: Продвинутые темы в C


Подмодуль содержит 3 задания повышенной сложности по работе с вложенными структурами, гибкими массивами и битовыми полями.


## Темы заданий


| № | Название | Тема |
|---|----------|------|
| 1 | Блок для самых умных: структура в структуре | Поиск максимума по полю вложенной структуры, возврат указателя на элемент массива |
| 2 | Блок для самых умных: гибкие массивы | Выделение памяти под структуру с гибким массивом через `malloc` и заполнение данными |
| 3 | Блок для самых умных: битовые поля | Упаковка и распаковка значений через структуру с битовыми полями (`pack`/`unpack`) |
