# Блок для самых умных: гибкие массивы


### Задание №2
- **Сложность:** сложно
- **Формулировка:**  
  Вам дано объявление структуры с гибким массивом (соответствует вашему варианту).  
  Напишите функцию `make_<name>`, которая:
  1. Принимает массив элементов и их количество `n`.
  2. Выделяет память под структуру с гибким массивом нужного размера через `malloc` (при неудачном `malloc` функция возвращает `NULL`.).
  3. Заполняет поле `count` значением `n` и копирует элементы в гибкий массив.
  4. Возвращает указатель на созданную структуру.

  Писать `main()` не нужно.

- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Объявление структуры (дано в условии) | Сигнатура функции | Тип элементов гибкого массива |
  |------------|---------------------------------------|-------------------|-------------------------------|
  | 0 | `typedef struct { int count; double values[]; } Stats;` | `Stats *make_stats(double *arr, int n)` | `double` |
  | 1 | `typedef struct { int count; int items[]; } IntList;` | `IntList *make_intlist(int *arr, int n)` | `int` |
  | 2 | `typedef struct { int count; float scores[]; } ScoreBoard;` | `ScoreBoard *make_scoreboard(float *arr, int n)` | `float` |
  | 3 | `typedef struct { int count; char letters[]; } CharBuf;` | `CharBuf *make_charbuf(char *arr, int n)` | `char` |
