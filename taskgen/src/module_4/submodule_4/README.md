# Подмодуль 4: Typedef в C


Подмодуль содержит 3 задания по использованию typedef для создания псевдонимов типов и упрощения объявления структур.


## Темы заданий


| № | Название | Тема |
|---|----------|------|
| 1.1 | Typedef: создание псевдонима | Создание псевдонима для целочисленного базового типа |
| 1.2 | Typedef: псевдоним и формат вывода вещественного типа | Создание псевдонима для вещественного типа и функция вывода с заданным форматом |
| 2 | Typedef: typedef struct | Объявление структуры через typedef и работа с объектом |
