# Подмодуль 4: Typedef в C

Подмодуль содержит 2 задания по использованию typedef для создания псевдонимов типов и упрощения объявления структур. Все задания на момент ококнчания второй итерации написаны только текстово.

## Темы заданий

| № | Название | Тема |
|---|----------|------|
| 1 | Typedef: создание псевдонима | Создание псевдонима для базового типа и выполнение арифметической операции |
| 2 | Typedef: typedef struct | Объявление структуры через typedef и работа с объектом |
