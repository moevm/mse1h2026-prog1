# Typedef: typedef struct

### Задание №2
- **Сложность:** легко  
- **Формулировка:**  
  Объявите структуру `{{ name }}`, описывающую `{{ entity }}`, используя синтаксис `typedef struct`.  
  Поля структуры:
  ```
  {{ fields_list }}
  ```
  Напишите программу, которая считывает из стандартного ввода данные одного объекта и выводит его поля через пробел.  
  **Требование:** тип должен быть объявлен именно через `typedef struct { ... } {{ name }};` — переменную объявляйте как `{{ name }} obj;`, а не `struct {{ name }} obj;`.  
  
  - **Параметры задания:**  
  Вариант структуры определяется значением `seed % 5`:
  
  | `seed % 5` | `name`     | `entity`       | Поля                                          | Формат вывода       |
  |------------|------------|----------------|-----------------------------------------------|---------------------|
  | 0          | `Student`  | студент        | `char name[32]`, `int age`, `float gpa`       | `%s %d %.1f`        |
  | 1          | `Book`     | книга          | `char title[64]`, `int pages`, `double price` | `%s %d %.2f`        |
  | 2          | `Point2D`  | точка          | `double x`, `double y`, `int label`           | `%.1f %.1f %d`      |
  | 3          | `Rectangle`| прямоугольник  | `double width`, `double height`, `char color[16]` | `%.1f %.1f %s`  |
  | 4          | `Employee` | сотрудник      | `char name[32]`, `int id`, `double salary`    | `%s %d %.2f`        |
  
  Строковые значения выбираются из фиксированного массива по индексу `seed % (размер массива)`. Сами массивы потом придумаю.
  
  - **Пример (для `seed = 10`, `seed % 5 = 0` → `Student`):**  
  
  **Входные данные:**
  ```
  Alice 20 2.0
  ```
  **Выходные данные:**
  ```
  Alice 20 2.0
  ```
