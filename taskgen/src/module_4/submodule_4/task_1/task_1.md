# Typedef: создание псевдонима

### Задание №1
- **Сложность:** легко  
- **Формулировка:**  
  Объявите псевдоним `{{ alias }}` для типа `{{ base_type }}`:  
  ```
  typedef {{ base_type }} {{ alias }};
  ```
  Напишите программу, которая считывает из стандартного ввода два значения типа `{{ alias }}` и выводит их {{ operation }}.  
  Формат вывода: `{{ fmt }}`  

- **Параметры задания:**  
  Вариант задания определяется значением `seed % 4`:

  | `seed % 4` | `base_type` | `alias`   | Операция          | `fmt`   |
  |------------|-------------|-----------|-------------------|---------|
  | 0          | `unsigned char` | `Byte`    | сумма             | `%hhu`  |
  | 1          | `float`     | `Real`    | сумма             | `%.2f`  |
  | 2          | `long long` | `BigNum`  | произведение      | `%lld`  |
  | 3          | `double`    | `Precise` | разность (a − b)  | `%.4f`  |


- **Пример (для `seed = 6`, `seed % 4 = 2` → `BigNum`):**  

  **Входные данные:**
  ```
  16 11
  ```
  **Выходные данные:**
  ```
  176
  ```
