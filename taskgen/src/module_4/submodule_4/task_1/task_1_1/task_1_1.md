### Задание №1.1 — псевдоним для целочисленного типа
- **Сложность:** легко  
- **Формулировка:**  
  Напишите **одну строку** — объявление псевдонима для целочисленного типа, соответствующего вашему варианту.  

- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Базовый тип (`base_type`) | Псевдоним (`alias`) |
  |------------|--------------------------|---------------------|
  | 0 | `unsigned char` | `u8` |
  | 1 | `unsigned short` | `u16` |
  | 2 | `unsigned long long` | `u64` |
  | 3 | `long long` | `i64` |

- **Пример для `seed = 2` (`seed % 4 = 2`):**
  ```c
  typedef unsigned long long u64;
  ```
