# Typedef: создание псевдонима


### Задание №1.1
- **Сложность:** легко
- **Формулировка:**  
  Напишите **одну строку** — объявление псевдонима для целочисленного типа (соответствует вашему варианту).


- **Параметры задания:**  
  Вариант определяется значением `seed % 4`:

  | `seed % 4` | Базовый тип | Псевдоним |
  |------------|-------------|-----------|
  | 0 | `unsigned char` | `u8` |
  | 1 | `unsigned short` | `u16` |
  | 2 | `unsigned long long` | `u64` |
  | 3 | `long long` | `i64` |
