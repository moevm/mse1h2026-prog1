# Структуры данных

### Задание №2
- Cложность: легкая.
- Задание: опишите структуру `Word`, предназначенную для хранения одного слова. Словом считается последовательность латинских букв и цифр.

Структура должна содержать:
1. динамический массив символов (`char *data`), в котором хранится слово;
2. целое число (`int length`), хранящее длину слова;
3. дополнительное поле (`int extra`), тип и значение которого зависят от глобальной переменной `seed`.
Программа должна реализовать следующие функции:
`Word read_word()` - считывает слово из ввода и заполняет структуру `Word`;
`void print_word_info(Word w)` - выводит длину слова и значение дополнительного поля.

Программа должна:
1. считать одно слово `W`;
2. сохранить его в структуру `Word`;
3. вычислить значение дополнительного поля;
4. вывести два значения: длину слова и значение дополнительного поля.
- Уникальным становится значение `seed`, определяющее дополнительное поле:

Тип дополнительного поля определяется следующим образом:

если `seed % 3 == 0` — поле хранит количество цифр в слове;

если `seed % 3 == 1` — поле хранит количество заглавных букв в слове;

если `seed % 3 == 2` — поле хранит количество строчных букв в слове.
- Ввод:
```
W
```
- Пример вывода для `seed=13`:

Ввод:
```
HeLlo123
```
Вывод:
```
8 2
```
