# Оператор switch

### Задание №1
- Cложность: легкая.
- Задание: напишите программу, которая выполняет одно из текстовых действий над введённой строкой. Номер действия выбирается с помощью оператора `switch`.
Каждое действие должно быть реализовано в виде отдельной функции.
Программа должна:
1. Прочитать целое число `op` (номер операции).
2. Прочитать строку `S`.
3. C помощью оператора `switch` выбрать и вызвать нужную функцию, передав ей строку `S`.
Возможные действия:

0 — функция `void reverseString(char *str)`, выводящая строку `str` в обратном порядке.

1 — функция `void repeatChars(char *str)`, выводящая строку `str`, повторив каждый символ `r` раз. Значение `r` вычисляется как `(seed % 3) + 1`.

2 — функция `void removeVowels(char *str)`, выводящая строку `str` без гласных букв (латиница, оба регистра). 

3 — функция `void shiftRight(char *str)`, выполняющая циклический сдвиг строки вправо на `k` символов и выводящая результат.  Значение `k` вычисляется как `(seed % 5) + 1`.
- Уникальными становятся значения `r`, `k`, зависящие от `seed`.
- Ввод:
```
op
S
```
- Пример вывода для `seed=13`: 

Ввод:
```
1
Hello
```
Вывод:
```
HHeelloo
```
