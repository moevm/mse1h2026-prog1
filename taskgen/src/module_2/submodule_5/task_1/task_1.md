## Тема 5: Статические и динамические библиотеки
-------
### Объектные файлы `*.o`

#### Задание №5.1.1
- Сложность: легко
- Задание: Дан исходный файл `{file}.c`. Напишите команду gcc, которая скомпилирует его в объектный файл `{file}.o` без перехода к этапу линковки.
- Какие параметры задания параметризуются: `{file}` (имя файла)
- Ответы:
  - `gcc -c {file}.c`
