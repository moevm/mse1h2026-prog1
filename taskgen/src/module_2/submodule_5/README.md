# Подмодуль 5: Статические и динамические библиотеки

Подмодуль посвящён работе со статическими библиотеками в языке Си. Рассматриваются объектные файлы (`*.o`) как промежуточный этап сборки, процесс создания архивов библиотек (`*.a`) с помощью утилиты `ar`, а также правила линковки с подключением внешних библиотек через флаги `-L` и `-l`. Особое внимание уделяется порядку аргументов в команде компилятора, так как линковщик обрабатывает их слева направо, разрешая неразрешённые символы только по мере их появления.

## Темы заданий

| № | Название | Тема |
|---|----------|------|
| 5.1.1 | Объектные файлы `*.o` | Написать команду `gcc` для компиляции исходника в объектный файл |
| 5.2.1 | Статические библиотеки `*.a` | Определить полное имя файла библиотеки по используемому флагу `-l` |
| 5.3.1 | Создание библиотеки | Написать команду `ar` для упаковки объектных файлов в архив `*.a` |
| 5.4.1 | Линковка с библиотекой | Составить команду `gcc` с корректным указанием пути (`-L`) и имени (`-l`) |
| 5.5.1 | Порядок линковки | Указать правильную последовательность исходного файла и флага библиотеки |
