## Тема 5: Статические и динамические библиотеки
-------
### Статические библиотеки `*.a`

#### Задание №5.2.1
- Сложность: легко
- Задание: При линковке используется флаг `-l{name}`. Какое полное имя файла (с префиксом и расширением) будет автоматически искать линковщик в указанных директориях для статической библиотеки?
- Какие параметры задания параметризуются: `{name}` (имя библиотеки без префикса/расширения)
- Ответы:
  - `lib{name}.a`
