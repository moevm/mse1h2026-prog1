## Тема 1: Этапы сборки
-------
### Компиляция

#### Задание №1.2.1
- Сложность: легко
- Задание: Какой флаг `gcc` останавливает сборку после этапа компиляции?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `-S`

#### Задание №1.2.2
- Сложность: легко
- Задание: Какое расширение получит файл после этапа компиляции?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `.s`

#### Задание №1.2.3
- Сложность: легко
- Задание: На каком этапе сборки выполняется действие: `{действие}`?
- Какие параметры задания параметризуются: 
`{действие}`: {проверяется синтаксис программы, выполняется оптимизация программы, генерируется ассемблерный код}
- Ответы:
  - Компиляция

#### Задание №1.2.4
- Сложность: средне
- Задание: Дан файл `{name}.c`. Напишите команду `gcc`, чтобы выполнить только этап компиляции и сохранить результат в `{name}.s`.
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - `gcc -S {name}.c -o {name}.s`

#### Задание №1.2.5
- Сложность: сложно
- Задание: Дан файл `{name}.c`. После команды `gcc -S {name}.c` получен `{result_file}`. Напишите команду, чтобы продолжить сборку и получить `{prog}`.
- Какие параметры задания параметризуются: `{name}`, `{result_file}`, `{prog}`
- Ответы:
  - `gcc {name}.s -o {prog}`
