# Подмодуль 1: Этапы сборки

Подмодуль посвящён изучению этапов компиляции программ на языке Си.
Рассматриваются: препроцессинг, компиляция, ассемблирование, линковка, объектные и исполняемые файлы.

## Темы заданий

| № | Название | Тема    |
|---|----------|---------|
| 1.1.3 | Препроцессинг | Определить, что действие относится к этапу препроцессинга |
| 1.2.3 | Компиляция | Определить, что действие относится к этапу компиляции |
| 1.3.3 | Ассемблирование | Определить, что действие относится к этапу ассемблирования |
| 1.4.2 | Линковка | Составить команду линковки объектных файлов в исполняемый файл |
| 1.5.2 | Объектный файл | Составить команду gcc для сборки `.o` без линковки |
| 1.6.2 | Исполняемый файл | Составить команду полной сборки `.c` в исполняемый файл |
