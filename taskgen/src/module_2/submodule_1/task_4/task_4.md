## Тема 1: Этапы сборки
-------
### Линковка

#### Задание №1.4.1
- Сложность: легко
- Задание: На каком этапе сборки выполняется действие: `{действие}`?
- Какие параметры задания параметризуются: 
`{действие}`: {создается таблицы релокаций, происходит связь вызовов функций и их определением, происходит релокация, объединяется итоговая секция}
- Ответы:
  - Линковка

#### Задание №1.4.2
- Сложность: средне
- Задание: Даны `{N}` файлов `{f1}.c, {f2}.c, ...`, уже скомпилированных в `.o`. Напишите команду линковки в `{prog}`.
- Какие параметры задания параметризуются: `{N}` (2–4), `{f1...fN}`, `{prog}`
- Ответы:
  - `gcc {f1}.o {f2}.o ... -o {prog}`

#### Задание №1.4.3
- Сложность: сложно
- Задание: Даны `{N}` файлов `.c`. Файл `{main}.c` вызывает функции из остальных. Напишите команды для раздельной компиляции и финальной линковки в `{prog}`.
- Какие параметры задания параметризуются: `{N}` (2–3), имена файлов, `{main}`, `{prog}`
- Ответы:
  - `N` команд вида `gcc -c {fi}.c -o {fi}.o`
  - Финальная команда: `gcc {f1}.o {f2}.o ... -o {prog}`

#### Задание №1.4.4
- Сложность: сложно
- Задание: Даны `{a}.c` и `{b}.c`. Файл `{a}.c` вызывает `{func}()`, определённую в `{b}.c`. Напишите 3 команды для раздельной сборки и линковки в `{prog}`.
- Какие параметры задания параметризуются: `{a}`, `{b}`, `{func}`, `{prog}`
- Ответы:
  - `gcc -c {a}.c -o {a}.o`
  - `gcc -c {b}.c -o {b}.o`
  - `gcc {a}.o {b}.o -o {prog}`
