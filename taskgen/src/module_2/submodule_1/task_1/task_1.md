## Тема 1: Этапы сборки
-------
### Препроцессинг

#### Задание №1.1.1
- Сложность: легко
- Задание: Какой флаг `gcc` останавливает сборку после этапа препроцессинга?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `-E`

#### Задание №1.1.2
- Сложность: легко
- Задание: Какое расширение получит файл после этапа препроцессинга?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `.i`

#### Задание №1.1.3
- Сложность: легко
- Задание: На каком этапе сборки выполняется действие: `{действие}`?
- Какие параметры задания параметризуются: 
`{действие}`: {раскрываются директивы, раскрываются макросы, обрабатывается условная компиляция, удаляются комментарии}
- Ответы:
  - Препроцессинг

#### Задание №1.1.4
- Сложность: средне
- Задание: Дан файл `{name}.c`. Напишите команду `gcc`, чтобы выполнить только этап препроцессинга и сохранить результат в `{name}.i`.
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - `gcc -E {name}.c -o {name}.i`

#### Задание №1.1.5
- Сложность: сложно
- Задание: Дан файл `{name}.c`. После команды `gcc -E {name}.c` получен `{result_file}`. Напишите команду, чтобы продолжить сборку и получить `{prog}`.
- Какие параметры задания параметризуются: `{name}`, `{result_file}`, `{prog}`
- Ответы:
  - `gcc {name}.i -o {prog}`
