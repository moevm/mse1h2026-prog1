### Объектный файл

#### Задание №1.5.1
- Сложность: легко
- Задание: Можно ли прочитать объектный файл `{name}.o` в текстовом редакторе?
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - Нет

#### Задание №1.5.2
- Сложность: средний
- Задание: Дан файл `{name}.c`. Напишите команду gcc, которая создаст объектный файл {name}.o без линковки.
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - gcc -c {name}.c -o {name}.o
