## Тема 1: Этапы сборки
-------
### Ассемблирование

#### Задание №1.3.1
- Сложность: легко
- Задание: Какой флаг `gcc` останавливает сборку после этапа ассемблирования?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `-c`

#### Задание №1.3.2
- Сложность: легко
- Задание: Какое расширение получит файл после этапа ассемблирования?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `.o`

#### Задание №1.3.3
- Сложность: легко
- Задание: На каком этапе сборки выполняется действие: `{действие}`?
- Какие параметры задания параметризуются: 
`{действие}`: {ассемблерная инструкция переводится в машинный код, создается структура объектного файла, формируется таблица символов}
- Ответы:
  - Ассемблирование

#### Задание №1.3.4
- Сложность: средне
- Задание: Дан файл `{name}.c`. Напишите команду `gcc`, чтобы выполнить только этап ассемблирования и сохранить результат в `{name}.o`.
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - `gcc -c {name}.c -o {name}.o`

#### Задание №1.3.5
- Сложность: сложно
- Задание: Дан файл `{name}.c`. После команды `gcc -c {name}.c` получен `{result_file}`. Напишите команду, чтобы продолжить сборку и получить `{prog}`.
- Какие параметры задания параметризуются: `{name}`, `{result_file}`, `{prog}`
- Ответы:
  -  `gcc {name}.o -o {prog}`
