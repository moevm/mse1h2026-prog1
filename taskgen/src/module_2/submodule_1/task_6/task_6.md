## Тема 1: Этапы сборки
-------
### Исполняемый файл

#### Задание №1.6.1
- Сложность: легко
- Задание: Как называется файл, создаваемый по умолчанию при полной сборке без `-o`?
- Какие параметры задания параметризуются: нет
- Ответы:
  - `a.out`

#### Задание №1.6.2
- Сложность: средне
- Задание: Дан файл `{name}.c`. Напишите одну команду `gcc` для полной сборки в исполняемый файл `{prog}`.
- Какие параметры задания параметризуются: `{name}`, `{prog}`
- Ответы:
  - `gcc {name}.c -o {prog}`
