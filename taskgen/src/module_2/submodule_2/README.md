# Подмодуль 2: Препроцессор

Подмодуль посвящён изучению директив и макросов препроцессора языка Си.
Рассматриваются: `#include`, `#define`, условная компиляция, операторы `#` и `##`, защита от повторного включения и типичные ошибки при работе с макросами.

## Темы заданий

| № | Название | Тема    |
|---|----------|---------|
| 2.1.4 | `#include`: `< >` и `""` | Выбрать корректный синтаксис подключения стандартных/собственных заголовков |
| 2.2.2 | `#define`: объектоподобный макрос | Написать директиву создания макроса-константы |
| 2.3.2 | `#define`: функциональный макрос | Написать макрос с аргументами для суммы |
| 2.4.1 | `#undef` | Определить поведение `#undef` для несуществующего макроса |
| 2.5.2 | `#if/#elif/#else/#endif` | Определить макрос значением по умолчанию, если он не был определён |
| 2.6.1 | `#error` | Составить условную логику с генерацией ошибки препроцессора |
| 2.7.1 | Опасности макросов | Вычислить результат макроса без скобок (приоритет операций) |
| 2.8.1 | Побочные эффекты/раскрытие макросов | Вычислить значение выражения через цепочку макроопределений |
| 2.9.2 | Многострочные макросы | Написать многострочный макрос |
| 2.10.1 | Безопасный макрос `do { ... } while (0)` | Написать безопасный многострочный `SWAP` |
| 2.11.1 | Оператор `#` (stringification) | Определить результат строкизации аргумента макроса |
| 2.12.1 | Оператор `##` (token pasting) | Определить результат склейки токенов |
