## Тема 2: Препроцессор
---------
### Директивы

#### define
##### Функциональный макрос

###### Задание №2.3.1
- Сложность: легко
- Задание: Является ли `#define {NAME} (x) ((x)*(x))` объектоподобным или функциональным макросом?
- Какие параметры задания параметризуются: `{NAME}`
- Ответы:
  - Функциональный

###### Задание №2.3.2
- Сложность: средне
- Задание: Напишите директиву `#define`, которая создаёт функциональный макрос `{NAME}`(a, b), возвращающий сумму аргументов.
- Какие параметры задания параметризуются: `{NAME}`
- Ответы:
  - `#define {NAME}(a, b) ((a) + (b))`

###### Задание №2.3.3
- Сложность: средне
- Задание: Напишите директиву `#define`, которая создаёт функциональный макрос `{NAME}`(x), возвращающий удвоенное значение аргумента
- Какие параметры задания параметризуются: `{NAME}`
- Ответы:
  - `#define {NAME}(x) ((x) * 2)`

###### Задание №2.3.4
- Сложность: сложно
- Задание: Напишите директиву `#define`, которая создаёт функциональный макрос `{NAME}`(a, b), возвращающий минимум из двух аргументов.
- Какие параметры задания параметризуются: `{NAME}`
- Ответы:
  - `#define {NAME}(a, b) ((a) < (b) ? (a) : (b))`
