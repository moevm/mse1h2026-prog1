## Тема 2: Препроцессор
---------
### Макрос

#### Конструкция do{...}while(0)
##### Задание №2.2.4
- Сложность: сложно
- Задание: Напишите безопасный многострочный макрос `SWAP({a}, {b})`, который обменивает значения переменных через временную переменную и использует конструкцию `do { ... } while (0)`.  
- Какие параметры задания параметризуются: `{a}` (a-z), `{b}` (a-z), a != b
- Эталонный ответ:
  - ```c
      #define SWAP({a}, {b}) \
          do { \
              int t = ({a}); \
              ({a}) = ({b}); \
              ({b}) = t; \
          } while (0)
    ```
