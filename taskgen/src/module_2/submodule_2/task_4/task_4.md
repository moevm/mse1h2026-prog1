### Директивы

#### ifdef и ifndef
##### Задание №2.1.12
- Сложность: легко
- Задание: Что делает директива `{#ifdef / #ifndef / #if / #undef}` для `{NAME}`?
- Какие параметры задания параметризуются: директива, `{NAME}`
- Выбор: a) включает блок кода, если макрос {NAME} определён
        b) отменяет ранее определённый макрос {NAME}
        c) включает блок кода, если макрос {NAME} не определён
        d) включает блок кода, если выражение {NAME} не равно нулю
- Ответы:
  - `#ifdef` -> включает блок кода, если макрос {NAME} определён
  - `#ifndef` -> включает блок кода, если макрос {NAME} не определён
  - `#if` -> включает блок кода, если выражение {NAME} не равно нулю
  - `#undef` -> отменяет ранее определённый макрос {NAME}

##### Задание №2.1.13
- Сложность: сложно
- Задание: Напишите директивы: если макрос `{NAME}` не определён - определить его значением `{val}`.
- Какие параметры задания параметризуются: `{NAME}`, `{val}`
- Ответы:
  - ```c
      #ifndef {NAME}
      #define {NAME} {val}
      #endif
    ```

##### Задание №2.1.14
- Сложность: сложно
- Задание: Напишите директивы: если `{NAME}` строго больше `{t1}` - определить макрос `{RESULT}` как `{a}`, если больше `{t2}` - как `{b}`, иначе - как `{c}`.
- Какие параметры задания параметризуются: `{NAME}`, `{RESULT}`, `{t1}`, `{t2}`, `{a}`, `{b}`, `{c}` (`t1 > t2`)
- Ответы:
  - ```c
      #if {NAME} > {t1}
      #define {RESULT} {a}
      #elif {NAME} > {t2}
      #define {RESULT} {b}
      #else
      #define {RESULT} {c}
      #endif
    ```
