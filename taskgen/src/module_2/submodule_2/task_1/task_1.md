## Тема 2: Препроцессор
---------
### Директивы

#### include
##### <> и ""
###### Задание №2.1.1
- Сложность: легко
- Задание: Где препроцессор ищет файл при `#include <{name}.h>`?
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - В системных директориях, где лежат библиотеки языка Си

###### Задание №2.1.
- Сложность: легко
- Задание: Где препроцессор сначала ищет файл при `#include "{name}.h"`?
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - В текущей директории

###### Задание №2.1.3
- Сложность: легко
- Задание: Дан файл `{name}.h`, которого нет в текущей директории. При `#include "{name}.h"` где препроцессор продолжит поиск файла?
- Какие параметры задания параметризуются: `{name}`
- Ответы:
  - В системных директориях

###### Задание №2.1.4
- Сложность: легко
- Задание: Какой синтаксис `#include` используется для подключения `{тип}` заголовочных файлов?
- Какие параметры задания параметризуются: `{тип}` - {стандартных/системных, собственных}
- Ответы:
  - Стандартные/Системные -> `<>`
  - Собственные -> `""`
