## Тема 4: Линковка
-------
### static на уровне файла

#### Задание №4.4.1
- Сложность: легко 
- Задание: В модуле с исполнительным файлом `{file}`.c требуется создать вспомогательную функцию `{func}`, принимающую параметр `{arg}` целочисленного типа и возвращающий остаток от удвоенного значения `{arg}` при делении на `{num}`. Функция должна быть видна только внутри текущего `*.c` файла и не экспортироваться линковщику. Напишите её полное определение.
- Какие параметры задания параметризуются: `{file}` (имя модуля), `{func}` (имя функции), `{arg}` (имя параметра), `{num}` (2-100)
- Ответы:
  - 
  ```c
    static int {func}(int {arg}) { return (2 * {arg}) % {num}; }
