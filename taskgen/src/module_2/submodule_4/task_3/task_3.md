## Тема 4: Линковка
-------
### Внутренняя линковка

#### Задание №4.3.1
- Сложность: легко 
- Задание: Напишите определение глобальной переменной `{var}` типа `{type}` со значением `{val}`, которая должна иметь внутреннюю линковку   (быть видимой только внутри текущего `*.c` файла и не экспортироваться линковщику).
- Какие параметры задания параметризуются: `{type}` (int, float, double), `{var}` (имя переменной), `{val}` (целое число) 
- Ответы:
  - `static {type} {var} = {val};`
