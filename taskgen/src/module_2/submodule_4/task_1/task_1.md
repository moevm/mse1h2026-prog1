## Тема 4: Линковка
-------
### Символы

#### Задание №4.1.1
- Сложность: легко 
- Задание:
Дан фрагмент кода:
```c
  int {var} = {val};
  void {func}(int a);
  #define {MAC} {num}
```
Перечислите через запятую все имена, которые линковщик будет обрабатывать как символы (в порядке появления в коде).
- Какие параметры задания параметризуются: `{var}` (имя переменной), `{func}` (имя функции), `{val}` (целое число), `{MAC}` (имя макроса), `{num}` (целое число)
- Ответы:
  - `{var}`, `{func}`
