### Задание №6
- Сложность: легко
- Задание: Что делает директива `{#ifdef / #ifndef / #if / #undef}` для `{NAME}`?
- Какие параметры задания параметризуются: директива, `{NAME}`
-Выбор: a) включает блок кода, если макрос {NAME} определён
        b) отменяет ранее определённый макрос {NAME}
        c) включает блок кода, если макрос {NAME} не определён
        d) включает блок кода, если выражение {NAME} не равно нулю
- Ответы:
  - `#ifdef` -> включает блок кода, если макрос {NAME} определён
  - `#ifndef` -> включает блок кода, если макрос {NAME} не определён
  - `#if` -> включает блок кода, если выражение {NAME} не равно нулю
  - `#undef` -> отменяет ранее определённый макрос {NAME}
