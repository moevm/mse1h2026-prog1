#### GNU-расширения

#### Задание №6.3.1
- Сложность: легко
- Задание: Какой флаг необходимо передать компилятору gcc, чтобы соответствовать стандарту языка C версии `{ver}` и при этом сохранить поддержку расширений GCC? Запишите флаг со значением.
- Какие параметры задания параметризуются: `{ver}` (11, 17)
- Ответы:
  - `-std=gnu{ver}`
