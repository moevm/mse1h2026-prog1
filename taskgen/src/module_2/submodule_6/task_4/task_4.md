## Тема 6: Компиляторы и флаги
-------
### Флаги оптимизации -O<число>

#### Задание №6.4.1
- Сложность: легко
- Задание: Дан файл `{file}.c`. Напишите команду gcc для его компиляции `{opt_desc}`. Запишите полную команду.
- Какие параметры задания параметризуются: `{file}` (имя файла),

`{opt_desc}`: {"без оптимизации, для отладки", "с локальной оптимизацией", "с оптимизацией баланса скорости и стабильности", "с максимальной оптимизацией производительности", "с оптимизацией по размеру кода"},

`{flag}`: {"-O0", "-O1", "-O2", "-O3", "-Os"} (порядок строго соответствует порядку `{opt_desc}`)

- Ответы:
  - `gcc {flag} {file}.c`
