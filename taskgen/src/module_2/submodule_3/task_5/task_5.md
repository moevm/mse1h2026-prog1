## Тема 3: Заголовочные файлы
-------
### inline

#### Задание №3.5.1
- Сложность: средний 
- Задание: Напишите безопасное для заголовочного файла определение функции `{func}`, которая принимает аргумент `{arg}` типа `{type}` и возвращает его квадрат `{arg}`. Используйте модификаторы, которые предотвратят ошибку множественного определения (multiple definition) при подключении заголовка в несколько `*.c` файлов.
- Какие параметры задания параметризуются: `{func}`, `{arg}`, `{type}`
- Ответы:
  - ```c
      static inline {type} {func}({type} {arg}) { return {arg} * {arg}; }
    ```
