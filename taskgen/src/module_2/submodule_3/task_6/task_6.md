## Тема 3: Заголовочные файлы
-------
### Что можно и нельзя писать в header

#### Задание №3.6.1
- Сложность: легко
- Задание: Выберите `{действие}` утверждения относительно правил размещения кода в заголовочных файлах в языке C?
- Какие параметры задания параметризуются:  `{действие}` - всего выбирается из 4 случайных утверждений из `{верные утверждения}` и `{неверные утверждения}`
- Выбор:
`{верные утверждения}`: {можно размещать объявления функций без тел, допускается определение макросов через #define, разрешено объявлять глобальные переменные с модификатором extern, безопасно описывать структуры и typedef, можно определять inline-функции при использовании static inline}
`{неверные утверждения}`: {можно определять глобальные переменные и выделять под них память в .h, разрешено писать тела обычных функций без модификаторов, следует объявлять static-переменные для общего состояния проекта, допустимо размещать исполняемые инструкции вне тел функций, можно подключать платформенные заголовки без #ifdef}

- Ответы: выражения проверяются на принадлежность соответствующему классу
