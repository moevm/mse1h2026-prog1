# Управляющие конструкции

### Задание №10
- Сложность: легкая.

- Задание: сколько чисел будет выведено на экран в результате работы следующей программы?
```
#include <stdio.h>

int main()
{
    int count = 0;
    for (int i = 1; i <= M; i++)
    {
        for (int j = 1; j <= N; j++)
        {
            if (i + j == S)
                continue;
            count++;
        }
    }
    printf("%d", count);
    return 0;
}
```   

- Уникальными становятся значения `M`, `N`, `S`.

- Ввод: пуст. Вывод при `M = 2`, `N = 2`, `S = 2`: `3`.
