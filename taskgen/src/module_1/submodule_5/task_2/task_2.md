# Выражения и операторы

### Задание №2
- Cложность: легкая.
- Задание: чему будет равно значение выражения `expression`?

- Уникальным становится значение `expression`, состоящее из целых и вещественных чисел, операторов `+`, `-`, `*`, `/`, `%` и явных привидений типов. 

- Ввод: пуст. Пример вывода при `expression = (float)13 / 2 + 7.5 - (int)3.14`: `11.0`.
