# Выражения и операторы

### Задание №2
- Cложность: легкая.
- Задание: чему будет равно значение выражения `expression`?

- Уникальным становится значение `expression`, состоящее из `целых и вещественных чисел, +, -, *, /, %`. 

- Ввод: пуст. Пример вывода при `expression = 13 + 7.4 * 2 - 7 / 3 + 5 % 2`: `26.8`.
