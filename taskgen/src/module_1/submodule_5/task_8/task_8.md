# Выражения и операторы

### Задание №8
- Cложность: средняя.
- Задание: есть выражение `expression`. Найдите значение этого выражения.

- Уникальным становится значение `expression`. Выражение состоит из целых чисел и `&, |, ^, ~, <<, >>`.

- Ввод: пуст. Пример вывода при `expression = 13 & 7 << 1`: `12`.
