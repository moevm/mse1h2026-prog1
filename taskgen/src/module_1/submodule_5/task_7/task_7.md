# Выражения и операторы

### Задание №7
- Cложность: легкая.
- Задание: есть выражение `expression`. Напишите логические операции в порядке их выполнения через пробел, затем через пробел укажите значение этого выражения.

- Уникальным становится значение `expression`. Выражение состоит из целых чисел, `+, -, *, /, %` и `&&, ||, !`.

- Ввод: пуст. Пример вывода при `expression = (17 - 5 * 3 - 2) || !(4 % 2) && 4`: `! && || 1`.
