# Выражения и операторы

### Задание №6
- Cложность: средняя.
- Задание: есть выражение `expression`. Напишите операции в порядке их выполнения (без пробела, скобки не считаются), затем через пробел укажите значение этого выражения.

- Уникальным становится значение `expression`. Выражение состоит из целых чисел, `+, -, *, /, %, &&, ||, !` и скобок.

- Ввод: пуст. Пример вывода при `expression = 2 * (7 + 5 % 3) - 4`: `%+*- 14`.
