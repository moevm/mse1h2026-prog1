# Выражения и операторы

### Задание №5
- Cложность: легкая.
- Задание: есть две переменные, хранящие целые числа: `const1` и `const2`, они уже инициализированы в программе. Напишите строку кода для нахождения `expression`. Ответ должен быть записан в переменной `answer`, которая раньше не встречалась.

- Уникальным становится значение `expression`: сумма, разность, произведение, остаток от деления, целая часть при делении.

- Ввод: пуст. Пример вывода при `expression = сумма`: `int answer = const1 + const2;`.
