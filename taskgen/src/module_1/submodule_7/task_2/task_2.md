# Функции и процедуры

### Задание №2
- Cложность: сложная.
- Задание: напишите функцию, которая переводит число из системы счисления с основанием `system` в десятичную систему счисления и возвращает результат. Функция имеет следующую сигнатуру: `int function(int value);`.

- Уникальными становится значения `system`. Это значение принадлежит диапозону от 2 до 9.

- Ввод: число в заданной системе счисления. Вывод: десятичное число.
