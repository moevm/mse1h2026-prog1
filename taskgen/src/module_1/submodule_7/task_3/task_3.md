# Функции и процедуры

### Задание №3
- Cложность: средняя.
- Задание: напишите функцию для проверки, является ли полученное натуральное число целой степенью числа `number`. Функция должна иметь следующую сигнатуру: `int function(int value);`. Если число является степенью, функция должна вернуть 1, иначе 0.

- Уникальными становится значения `number` - натуральное число, основание степени.

- Ввод: число. Вывод: ответ, является ли число какой-нибудь натуральной степенью числа `number`.
