# Функции и процедуры

### Задание №1
- Cложность: сложная.
- Задание: напишите процедуру, которая переводит число из десятичной системы счисления в систему счисления с основанием `system` и выводит результат на экран. Процедура имеет следующую сигнатуру: `void function(int value);`.

- Уникальными становится значения `system`. Это значение принадлежит диапозону от 2 до 9.

- Ввод: десятичное чило. Вывод: число в заданной системе счисления.
