# Структура программы в си

### Задание №7
- Сложность: легкая.

- Задание: дана программа. Определите, что она выведет на экран (два числа через пробел).
```
#include <stdio.h>

int main()
{
    int a = const1;
    int b = const2;
    a = a + b;
    b = a + b;
    a = a + b;
    printf("%d %d", a, b);
    return 0;
}
```

- Уникальными становятся значения `const1`, `const2`.

- Ввод: пуст. Вывод при `const1 = 13`, `const2 = 7`: `47 27`.
