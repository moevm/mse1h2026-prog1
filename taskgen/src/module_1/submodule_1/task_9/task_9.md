# Структура программы в си

### Задание №9
- Сложность: сложная.

- Задание: ниже приведен код программы, записанный без соблюдения правил форматирования. Перепишите его, строго придерживаясь указанного стиля:

1. отступы - 4 пробела, табуляции не используются;

2. открывающая фигурная скобка блоков - на новой строке (Allman);

3. после открывающей скобки блока не должно быть пустой строки.

Прочие детали (пробелы внутри скобок, длина строк и т.п.) можно не менять, если они не противоречат перечисленным требованиям.
```
#include <stdio.h>

int main(){int NAME = VAL;if (NAME > COND){printf(FMT, NAME);}else{printf("small");}return 0;}
```

- Уникальными становятся значения `VAL`, `COND`, `NAME`, `FMT`.

- Ввод: пуст. Вывод при `VAL = 14`, `COND = 16`, `NAME = value`, `FMT = big number %d`:
```
#include <stdio.h>

int main()
{
    int value = 14;
    if (value > 16)
    {
        printf("big number %d", value);
    }
    else
    {
        printf("small");
    }
    return 0;
}
```
