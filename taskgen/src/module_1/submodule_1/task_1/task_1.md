# Структура программы в си

### Задание №1
- Cложность: легкая.
- Задание: в текущей директории находится файл src_name.c, который выводит на экран сообщение "Hello, world!". Какую команду нужно ввести для компиляции, чтобы исполняемый файл назывался src_name.

- Уникальными становятся значения `src_name`.

- Ввод: пуст. Вывод: `gcc src_name.c -o src_name` или `gcc -o src_name src_name.c`.
