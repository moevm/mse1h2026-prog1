# Структура программы в си

### Задание №4
- Cложность: средняя.
- Задание: напишите объявление функции по ее определению.
```
    type function(type1 obj1, type2 obj2, type3 obj3)
    {
        type obj4 = answer;
        return answer;
    }
```

- Уникальными становятся значения `type, type1 - type3, answer`.

- Ввод: пуст. Пример вывода при `type = int, type1 = char, type2 = int, type3 = long, answer = 13`: `int function(type1 obj1, type2 obj2, type3 obj3);`.
