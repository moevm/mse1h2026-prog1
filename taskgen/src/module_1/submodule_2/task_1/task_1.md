# Базовые типы данных

### Задание №1
- Cложность: легкая.
- Задание: какие из ниже представленных выражений являются истинными? Ответы укажите через пробел.
```
1. expression1
2. expression2
3. expression3
4. expression4
5. expression5
6. expression6
7. expression7
8. expression8
```

- Уникальными становятся значения `expression1 - expression8`. Хотя бы одно выражение истинно. Порядок также зависит от `seed`. Выражения `expression1 - expression8` могут быть числом, зависящим от `seed`, выражением с `+` и/или `-`, `NULL`.

- Ввод: пуст. Вывода при:
1. 13
2. 7 - 7
3. NULL
4. -3 + 1
5. 16
6. 7
7. 6 + 7
8. 0
`Ответ: 1 4 5 6 7.`
