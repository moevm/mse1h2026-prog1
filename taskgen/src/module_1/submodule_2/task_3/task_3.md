# Базовые типы данных

### Задание №3
- Cложность: средняя.
- Задание: в программе нужно хранить число `number` в переменной value. Выбирите наиболлее подходящий тип данных (float, double, long double) для хранения этого числа. Помните про разные затраты памяти и разные точности у форматов. Напишите строку кода для объявление переменной value, присвойте ей значение `number`. 

- Уникальными становятся значения `number`.

- Ввод: пуст. Пример вывода при `number = 3.14`: `float value = 3.14;`.
