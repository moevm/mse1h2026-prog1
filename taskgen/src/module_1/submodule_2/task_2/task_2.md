# Базовые типы данных

### Задание №2
- Cложность: средняя.
- Задание: чему равно значение выражения `expression`, если его вывести с помощью спецификатора `spec`?

- Уникальными становятся значения `expression` и `spec`. Выражение состоит из символьного литерала и целого числа (операция `+` или `-`), например: `'a' + 11` или `'v' - 13`. Спецификатор `spec` определяет формат вывода:
1. `%c` - символ
2. `%d` - десятичное целое число
3. `%x` - шестнадцатеричное целое число
4. `%o` - восьмеричное целое число
5. `%b` - двоичное целое число

- Ввод: пуст. Пример вывода при `expression = 'b' + 13` и `spec = %x`: `6f`.
