# Базовые типы данных

### Задание №2
- Cложность: средняя.
- Задание: чему равно значение выражения: `expression`?

- Уникальными становятся значения `expression`. Выражение состоит из символьного литерала и числа, например: `'a' + 11` или `'v' - 13`.

- Ввод: пуст. Пример вывода при `expression = 'b' + 13`: `o`.
