# Базовые типы данных

### Задание №4
- Сложность: средняя.

- Задание: определите, какие числа будут выведены на экран после выполнения следующей программы. Числа напишите через пробел.
```
#include <stdio.h>

int main() {
    signed char a = const;
    unsigned char b = const;
    a = a * 2;
    b = b * 2;
    printf("%d %d", (int)a, (int)b);
    return 0;
}
```

- Уникальными становятся значения `const`.

- Ввод: пуст. Вывод при `const = 127`: `-2 254`.
