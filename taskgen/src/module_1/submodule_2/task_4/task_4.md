# Базовые типы данных

### Задание №4
- Cложность: легкая.
- Задание: истинно ли следующее выражение: `expression`? Ответ: да или нет. 

- Уникальными становятся значения `expression`. Оно состоит из двух вещественных чисел и знака `>, <, ==`.

- Ввод: пуст. Пример вывода при `expression = 0.999999999999999999999 == 1.0`: `да`.
