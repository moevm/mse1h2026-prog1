# Базовые типы данных

### Задание №5
- Сложность: легкая.

- Задание: ниже представлен фрагмент программы. Определите, что он выведет. 
```
unsigned short s = const;
s = s * MUL;
printf("%hu", s);
```

- Уникальными становятся значения `const`, `MUL`.

- Ввод: пуст. Вывод при `const = 13131`, `MUL = 13`: `39631`.
