# Базовые типы данных

### Задание №11
- Сложность: легкая.

- Задание: дана программа, которая выводит на экран строку:
```
text
```
Программа:
```
#include <stdio.h>

int main() {
    printf("Ваша строка");
    return 0;
}
```
Определите, что должно быть на месте `Ваша строка`, чтобы программа выводила строку, представленную выше. Используйте эскейп последовательности.   

- Уникальными становятся значения `text`. Оно состоит из `word1`, `symbol`, `word2`. `symbol` - это один из следующих символов: перенос строки, табуляция, обратный слеш, двойные кавычки.

- Ввод: пуст. Вывод при `text = Hello " world`: `Hello \" world`.
