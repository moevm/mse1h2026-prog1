# Константы и летералы

### Задание №1
- Cложность: легкая.
- Задание: чему будет равно значение выражения `expression`? Часть программы с определением констант:
```
#define A value1
#define B value2
#define C value3
```

- Уникальными становятся значения `A, B, C` и `expression`, состоящее из `A, B, C, +, -, *`. 

- Ввод: пуст. Пример вывода при `A = 13, B = 3, C = 7`, `expression = A + 4 * B - 3 * C * A`: `-248`.
