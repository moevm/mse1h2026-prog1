# Константы и летералы

### Задание №1
- Cложность: средняя.
- Задание: чему будет равно значение выражения `expression`? Часть программы с определением макросов:
```
#define A value1
#define B value2
#define C value3
```

- Уникальными становятся значения `value1`, `value2`, `value3` (арифметические выражения без скобок) и `expression`, состоящее из `A, B, C, +, -, *`. 

- Ввод: пуст. Пример вывода при `value1 = 1+2`, `value2 = 3*4`, `value3 = 5-6`, `expression = A + B * C`: `57`.
