# Константы и летералы

### Задание №3
- Cложность: средняя.
- Задание: напишите, как выглядит число `number` в привычной десятичной форме. Число записано в экспоненциальной форме и содержит дробную часть. Например: `2.5e-2 = 0.025`.

- Уникальными становятся значения `number`. Это число содержит экспоненту в записи и имеет десятичную точку.

- Ввод: пуст. Пример вывода при `number = 2.5e1 + 3.2e-2`: `25.032`.
