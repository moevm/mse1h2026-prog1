# Константы и летералы

### Задание №2
- Cложность: легкая.
- Задание: напишите, как выглядит число `number` в привычной форме. Например: `1e2 =  100`.

- Уникальными становятся значения `number`. Это число содержит экспоненту в записи.

- Ввод: пуст. Пример вывода при `number = 5e2 + 1e4`: `10500`.
