# Ввод и вывод

### Задание №2
- Сложность: средняя.
- Задание: напишите программу, которая выводит на экран символьную фигуру - прямоугольную сетку высотой `h` и шириной `w`. Для каждого элемента в позиции `(i, j)`, где `i` - номер строки (сверху вниз, начиная с 0), а `j` - номер столбца (слева направо, начиная с 0), программа должна напечатать символ по правилу:
  - если `(i + j + c) % a == 0`, вывести символ `s`;
  - иначе если `(i + j + c) % 2 == 0`, вывести символ `t`;
  - иначе вывести пробел (`' '`).
  Столбцы разделяются символом табуляции. 

- Уникальными становятся значения `h`, `w`, `c`, `a`, `s`, `t`.

- Ввод: пуст. Вывод: символьная сетка размера `h` на `w`, сформированная по указанному правилу.
