# Ввод и вывод

### Задание №2
- Cложность: средняя.
- Задание: напишите программу, которая рисует прямоугольник размером `h на w`. Для каждого `ij` элемента: если `(i + j + c) % a == 0`, тогда на его место вывести символ `s`, иначе если `(i + j + c) % 2 == 0`, тогда вывести символ `t`, иначе вывести пробел.

- Уникальными становится значения `h, w, c, a, s, t`.

- Ввод: пуст. Вывод: прямоугольник размера `h на w`.
