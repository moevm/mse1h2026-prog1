### Задание №4
- Сложность: легкая.

- Задание: что именно напечатает фрагмент программа? Запишите три символа (или числа) через пробел, которые появятся на экране.
```
int a = A;
printf("%d %c %x", a, a, a);
```

- Уникальными становятся значения `A`.

- Ввод: пуст. Вывод при `A = 65`: `65 A 41`.
