# Ввод и вывод

### Задание №1
- Cложность: легкая.
- Задание: напишите, что выведет следующий фрагмент программы:
    int a = const;
    printf("%flag", a);
- Уникальными становится значения const, flag.
const = seed % 20 + 10
seed % 2 == 0: flag = o
seed % 2 == 1: flag = x
- Ввод: пуст. Пример вывода для seed=13: const = 23, flag = x. Ответ: 17.
